# 🛡️ VanderGuard Ultimate
> **Premium Lua Protection & Distribution Matrix**

VanderGuard is a high-fidelity, industrial-grade script protection system inspired by LuArmor. It provides robust virtualization, licensing infrastructure, and real-time telemetry for Lua script developers.

## 💎 Elite Features
*   **bytecode Virtualization**: Convert scripts into custom VM-executable bytecode (v3.1 Engine).
*   **License Matrix**: Automated Batch Key Generation (1 Day, 1 Week, 30 Days, Lifetime).
*   **Dynamic Key GUI**: Automated, branded in-game GUI that handles HWID locking and activation.
*   **Restriction Kernels**: Triple-layer protection (IP, HWID, and UserId blacklisting).
*   **Network Pulse**: Real-time dashboard telemetry via Chart.js integration.
*   **Security Protocol**: Anti-Dump Matrix with 700KB corruption bait and header spoofing detection.

## 🚀 Deployment
VanderGuard is designed to be hosted "everywhere." 

### One-Click Vercel (Recommended)
1. Connect your GitHub account to [Vercel](https://vercel.com).
2. Link the repository.
3. Your industrial dashboard is live!

### One-Click Koyeb
1. Connect to [Koyeb](https://koyeb.com).
2. Link the repo and set the port to `4444`.

## 🛠️ Usage
1. Use the **Project Vault** to upload and protect your `.lua` assets.
2. Toggle the **Key System** on or off per script.
3. Copy the **Loader** and distribute it. 
4. **BOOM.** Your script is now protected by the Vander Network.
